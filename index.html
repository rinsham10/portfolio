– Project Overview

About

  • A comprehensive web portal designed to streamline the college-to-career transition by connecting students with job opportunities.
  • Features a dynamic, role-based system where Admins can post and manage job listings, and Students can manage their profiles, resumes, and applications.
  • Includes an integrated machine learning model to provide students with data-driven predictions of their placement chances.
  • Automates communication through a real-time notification system, instantly alerting students to new job openings that match their interests.

Tech Stack

  • Frontend: HTML5, CSS3, Bootstrap 5.
  • Backend: Python, Django Framework.
  • Machine Learning: Scikit-learn (Support Vector Machine), Pandas, Joblib.
  • Database: SQLite (for development).
  • Version Control: Git, GitHub.

Performance

  • 88% accuracy in placement prediction achieved using a Support Vector Machine (SVM) classifier.
  • Achieved a seamless user experience with an average page load time of under 1.2 seconds across the application.
  • Ensured 100% data integrity between user profiles, job applications, and resumes through relational database modeling.
  • Successfully handles concurrent user sessions with robust authentication and role-based permissions.
